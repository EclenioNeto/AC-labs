# Projeto em Assembly usando o módulo PEPE - Pac-Man

## Descrição
Este projeto foi desenvolvido em Assembly para o módulo **PEPE**.
O objetivo é desenhar e mover um boneco numa matriz de pixels
(32x32), usando acesso direto à memória de vídeo (pixelScreen).

## Funcionalidades
- Desenho de pixels manualmente
- Movimento do pac-man, em todas as direções, pelo teclado
- Moldura fixa no ecrã
- Comidas fixas na tela
- Limitação dos movimentos nas extremidades da pixelScreen

## Conceitos usados
- Endereçamento de memória
- Manipulação de bits
- Matrizes de pixels
- Rotinas em Assembly

## Como executar
1. Abrir o módulo PEPE no simulador (Architcture Simulator 2019)
2. Carregar o ficheiro `.asm`
3. Executar o programa
4. Abrir o pixelScreen para visualizar a execução
5. Usar o teclado para mover o boneco

## Requisitos
- Simulador PEPE
- Conhecimentos básicos de Assembly
